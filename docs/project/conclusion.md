# Висновки

Цей проект є результатом розробки ефективної системи управління проєктами, націленої на створення основних функціональних можливостей. Проведене тестування підтвердило працездатність ключових операцій, таких як створення ролей, користувачів, команд, проєктів, завдань, артефактів та призначення грантів. Успішно реалізовано пошук користувачів та проєктів за відповідними критеріями.
Під час розробки моделі даних було приділено увагу забезпеченню цілісності та ефективності збереження інформації. Використання програмного коду на Python для безпосередньої взаємодії з базою даних дозволило продемонструвати основні операції створення та пошуку, підтверджуючи коректність реалізації DAO (Data Access Object) шару та відповідність моделі даних поставленим завданням.
Результати тестування свідчать про те, що розроблена система здатна виконувати основні функції управління проєктами. Проект демонструє практичне застосування принципів проєктування баз даних та вміння створювати працездатні програмні рішення.
